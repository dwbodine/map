# Marie Anna Photography

This is the business website for Marie Anna Photography - marieannaphotography.com

Interesting ponts:
- React 19+ with Typescript hosted in an static S3 bucket on AWS
- Uses Webpack instead of NextJS (or any other framework)
- no database is used - data is static Typescript
- uses `react-photo-album` to display photos in a carousel
- used to be hosted on GoDaddy for $120+ per year, now site in AWS for $0.60/month
